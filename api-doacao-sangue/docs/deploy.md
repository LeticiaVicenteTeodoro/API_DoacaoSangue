# Deploy

A API foi preparada para execução em ambiente de produção.

O deploy será realizado utilizando uma plataforma de hospedagem em nuvem.

Etapas:

- Instalação das dependências

- Configuração das variáveis de ambiente

- Inicialização do banco

- Execução do scheduler

- Inicialização do servidor FastAPI
